# TRUE Simulation Project

This directory contains the executable approximation of the TRUE simulation experiment.

## Structure

- `src/`: core simulation code
- `scripts/`: helper scripts for running experiments and generating reports
- `docs/`: experiment design documents and project notes
- `data/`: experiment outputs, grouped by run family
- `reports/`: curated report deliverables
- `assets/`: static assets such as fonts

## Main Entry Points

- `src/true_simulation.py`
- `scripts/run_true_experiment.sh`
- `scripts/generate_true_report_pdf.py`

## Notes

- `data/final_results_v2/` is the current primary result set.
- `reports/final/` contains the polished Markdown and PDF report artifacts derived from the primary result set.
